# Sales Analytics Business Intelligence Dashboard

## Overview
End-to-end Data Analytics project demonstrating SQL, Python, KPI design, business intelligence, and dashboard planning.

## Business Objective
Analyze sales performance across regions, products, and customers to identify growth opportunities and profitability drivers.

## Tech Stack
- SQL
- Python (Pandas)
- Power BI
- Excel

## KPIs
- Revenue
- Profit
- Profit Margin
- Average Order Value
- Regional Performance

See folders for SQL, Python analysis, data, and reporting assets.
